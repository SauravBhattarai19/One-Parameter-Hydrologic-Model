qgisMinimumVersion=3.16
description=Variable Source Area – One Parameter Model (VSA-OPM) distributed hydrologic model. Runs DEM preprocessing, precipitation (gauge or IMERG satellite), VSA/Green-Ampt/impervious runoff generation, and kinematic- or diffusive-wave routing inside QGIS with optional GPU (CuPy/CUDA) acceleration.
version=2.0.0
author=ORISE / Saurav Bhattarai
email=sauravbhattarai@example.com

about=This plugin wraps the full VSA-OPM pipeline (Pradhan & Ogden 2010) for use directly inside QGIS. It supports:
  - DEM reprojection, sink-filling, flow-direction/accumulation, and watershed delineation via pysheds
  - Five precipitation engines: uniform, Thiessen, IDW, and NASA GPM IMERG V07 satellite (Thiessen/IDW) via Google Earth Engine
  - Runoff generation modes: none, runoff coefficient, pre-computed raster, SCS-CN, and VSA-OPM with composable mechanisms (saturation-excess VSA + Green-Ampt infiltration-excess + urban impervious shedding)
  - Optional SERVES satellite soil-moisture deficit, SoilGrids porosity/texture, and HiHydroSoil vertical Ksat via Earth Engine
  - Kinematic- or diffusive-wave (CASC2D/GSSHA-style) routing with confined channel cross-sections and adaptive CFL time-stepping
  - Spatial Manning's n from ESA WorldCover (LULC) or WUDAPT LCZ, plus per-run mass-balance reporting
  - Optional GPU acceleration via CuPy/CUDA (falls back to CPU automatically)
  - QGIS Processing Framework integration for scripting and the Graphical Modeller
